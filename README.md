<h2>📖 Core Mission</h2>
<p>
  This repository hosts a suite of research projects dedicated to the convergence of <b>Artificial Intelligence</b> and <b>Wireless Communications</b>. As we move toward 6G, the goal is to shift from "AI-aided" systems to <b>AI-native architectures</b> where machine learning is embedded within the physical layer, radio access network (RAN), and network management plane.
</p>

<hr>

<h2>🧠 Research Pillars</h2>

<h3>1. Intelligent Signal Processing & AMC</h3>
<p>
  Focuses on the automation of signal intelligence. Our flagship project in this pillar utilizes <b>Hybrid ResNet-LSTM</b> architectures to perform <b>Automatic Modulation Classification (AMC)</b>. 
</p>
<ul>
  <li><b>Spatial Feature Extraction:</b> Using Deep Residual Learning to identify complex patterns in I/Q signal components.</li>
  <li><b>Temporal Dependencies:</b> Leveraging LSTM layers to model the time-varying nature of wireless channels.</li>
  <li><b>Robustness:</b> Optimizing classification accuracy under low SNR and high-interference environments.</li>
</ul>

<h3>2. Autonomous Network Management & Agentic AI</h3>
<p>
  Investigating the use of <b>Autonomous AI Agents</b> for 6G network orchestration. This includes:
</p>
<ul>
  <li><b>Self-Organizing Networks (SON):</b> Implementing agents for self-healing and self-optimization.</li>
  <li><b>Intelligent Resource Allocation:</b> Using Reinforcement Learning for dynamic spectrum access.</li>
  <li><b>Semantic Communication:</b> Moving beyond bit-perfect transmission toward goal-oriented, semantic-aware data exchange.</li>
</ul>
